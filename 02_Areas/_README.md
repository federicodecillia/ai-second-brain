# Choosing your areas

PARA's Areas are your ongoing responsibilities with no end date (Projects, by contrast, have deadlines).

Create a folder here **only for areas you actually have**. Do not pre-create empty folders.

Examples by vertical:
- Consultant / freelance: `clients`, `marketing`, `operations`
- Lawyer: `active-cases`, `clients`, `practice`
- Designer: `clients`, `portfolio`, `craft`
- Academic: `research`, `teaching`, `advising`

To create an area: make a folder and add a MOC note using `03_Resources/templates/area.md`.
