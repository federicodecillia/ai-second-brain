## For future agents
Task cockpit generated by the Obsidian **Tasks** plugin. It aggregates every `- [ ]` task across the vault (hub, routines, project/area files) with its emoji metadata. Do not write tasks here: write them in the right file (a project/area file, loose -> [[hub]], recurring -> [[routines]]) and they appear here. Requires the Tasks plugin.

> [!info] Emoji convention: `📅` due · `⏫`/`🔺` high priority · `🔽` low · `🔁` recurring · `✅` done

## 🔴 Overdue
```tasks
not done
due before today
sort by due
```

## 📅 Today and next 7 days
```tasks
not done
due before in 8 days
due after yesterday
sort by due
sort by priority
```

## ⏫ Prioritized, no due date
```tasks
not done
no due date
(priority is above none)
sort by priority
```

## 🗂 All open tasks, grouped by file
```tasks
not done
group by filename
sort by due
```

## ✅ Recently completed (last 14 days)
```tasks
done
done after 14 days ago
sort by done reverse
```
